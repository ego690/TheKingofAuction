# The King of Auction

A warehouse auction game prototype.

## Web Prototype

Open the static web prototype directly in your browser:

```text
D:\TheKingofAuction\web\index.html
```

The web prototype currently focuses on the warehouse auction loop: circular viewing holes, AI bidding, player tools, item reveal, settlement, and codex unlocks.

## Open in Godot

Use the Godot editor at:

```text
D:\steam\steamapps\common\Godot Engine\godot.windows.opt.tools.64.exe
```

Open this folder as the project:

```text
D:\TheKingofAuction
```

The main scene is `res://scenes/main.tscn`.
